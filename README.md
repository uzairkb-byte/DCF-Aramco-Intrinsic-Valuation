# DCF-Aramco-Intrinsic-Valuation
Discounted Cash Flow (DCF) intrinsic valuation model for Saudi Aramco (2222.SR) — covering historical financials (FY2021–FY2025), WACC build, five-year FCF projections, terminal value (GGM + EV/EBITDA exit multiple), sensitivity analysis, and a SELL recommendation with a ~$3.90 price target implying ~38% downside to the April 2026 market price.
